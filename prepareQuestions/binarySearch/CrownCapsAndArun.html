Arun has a hobby of crowncaps collection and he owns many crowncaps. Today, he gotN crowncaps and decided to form a pyramid using these crowncaps.

Triangle formation is given as follows:

  • First row can only have one crowncap.
  • 2nd row can only have two crowncaps.
  • 3rd row can only have three crowncaps.
  • and so on as shown in the given figure.

The pyramid would look something like this:

*

**

***


Arun wants to form a pyramid with maximum possible height using at most N crowncaps. He is not so good with maths so can you help him in finding the maximum height of the pyramid?

Input

The first line of input contains a single integer T denoting the number of test cases.

The first and the only line of each test case contains an integer N denoting the number of crowncaps Arun has.

Output

For each test case, output a single line containing an integer corresponding to the maximum possible height of the pyramid that Arun can get.

Constraints

  • 1 ≤ T ≤ 100
  • 1 ≤ N ≤ 109

Example

Input
3
4
5
7

Output
2
2
3

Explanation

  • Test 1: Arun can't form a pyramid with height > 2 as it requires atleast 6 crowncaps.
  • Test 2: Arun can't form a pyramid with height > 2 as it requires atleast 6 crowncaps.
  • Test 3: Arun can't form a pyramid with height > 3 as it requires atleast 10 crowncaps.
